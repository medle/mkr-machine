После импорта проекта из Arduino в Atmel Studio необходимо исправить
в Project properties/Linker:
  -Tflash_with_bootloader.ld

Если сделать Upload c другим файлом .ld то COM порт перестанет определяться.
Надо будет вручную перевести плату в режим программирования двумя нажатиями
кнопки Reset и загрузить ее из Arduino IDE с правильным загрузчиком.
